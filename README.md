# SuperHeroFights
Generate random fights using superhero API

## Quick Start

Clone repo and go to the source directory, and then copy paste this snippet:

```python
from battle import Battle

battle = Battle()
battle.create_teams()
battle.prepare_teams_to_fight()
battle.start()
```
